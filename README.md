## Design Philosophy

The website's aesthetic reflects the studio's guiding principle: **"Beauty in the essential"**. It features a minimalist, high-end visual style characterized by:

  * **Minimalist Palette:** A refined color scheme using "Oatmeal," "Sage," and "Charcoal."
  * **Typography-Forward:** Utilization of *Cormorant Garamond* for headers and *DM Sans* for body text to create a sophisticated editorial feel.
  * **Functional Elegance:** A focus on the intersection of restraint and intention, inspired by both the landscapes of Giza and functional design.

## Project Features

  * **Interactive Gallery:** A "Collection" section showcasing selected works through a grid layout with hover effects and detailed modal views.
  * **Smooth Navigation:** Implements smooth-scroll behavior for internal links (Work, About, Contact).
  * **Modern UI Components:** Includes a custom-built modal system for project details and a noise-textured overlay for added depth.
  * **Responsive Design:** Fully optimized for different screen sizes, including mobile-specific layouts.
  * **Micro-animations:** Subtle fade-in animations triggered on scroll using the Intersection Observer API.

## Selected Works Included

The portfolio showcases various design disciplines, including Product Design, Campaign Design, and Editorial Design. Key featured projects include:

  * **Apple Watch Series 10:** A study in precision and product design.
  * **AirTag Campaign:** "Lose Your Knack," an exercise in minimalist wit.
  * **MacBook Air M2 Visual:** A luxury tech visual rendered in violet and midnight tones.
  * **AirPods Max Study:** A product visual focusing on the "Graphite" edition.

## Technical Stack

  * **HTML5:** Semantic structure for the portfolio layout.
  * **CSS3:** Custom properties (variables), Grid, Flexbox, and keyframe animations.
  * **JavaScript (Vanilla):** Intersection Observer for scroll animations and custom modal logic.
  * **Google Fonts:** Integration of *Cormorant Garamond* and *DM Sans*.
